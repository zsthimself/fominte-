## 🔍 常见问题排查

### 元字段不显示

1. 检查元字段是否在后台定义
2. 检查命名空间是否为 `custom`
3. 检查字段名拼写是否正确
4. 检查文章是否填充了数据

### 分隔符解析错误

1. 检查数据中是否意外包含 `||` 或 `??`
2. 检查 `split()` 参数是否正确
3. 在 trim() 前检查是否有空项

### CSS 样式不生效

1. 检查选择器权重（使用浏览器开发者工具）
2. 检查是否被主题样式覆盖
3. 尝试添加 `!important`（谨慎使用）
