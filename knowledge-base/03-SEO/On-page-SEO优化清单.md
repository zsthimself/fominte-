# On-page SEO 优化清单

> 最后更新：2026-01-08

---

## 核心概念

> [!IMPORTANT]
> 仅仅在正文里"提到"关键词是不够的。搜索引擎会看**多个位置**，每个位置的权重不同。

---

## 关键词布局位置（按权重排序）

| 位置 | 权重 | 说明 |
|-----|------|------|
| **Title 标签** | ⭐⭐⭐⭐⭐ | 最重要！必须包含核心关键词 |
| **H1 标题** | ⭐⭐⭐⭐ | 页面主标题，与 Title 呼应 |
| **URL** | ⭐⭐⭐⭐ | 简短、包含关键词、用连字符分隔 |
| **Meta Description** | ⭐⭐⭐ | 影响点击率，间接影响排名 |
| **正文首段** | ⭐⭐⭐ | 开头 100 字内自然出现关键词 |
| **H2/H3 小标题** | ⭐⭐ | 使用相关词、变体词 |
| **正文内容** | ⭐⭐ | 自然分布，不要堆砌 |
| **图片 Alt** | ⭐⭐ | 图片描述包含关键词 |
| **内链锚文本** | ⭐⭐ | 从其他页面链接过来的文字 |

## 示例：优化 "lace fabric wholesale" 页面

```html
<!-- Title 标签 (50-60字符) -->
<title>Lace Fabric Wholesale | Bulk Bridal Lace Supplier - Fominte</title>

<!-- Meta Description (150-160字符) -->
<meta name="description" content="Buy lace fabric wholesale from Fominte. 
Premium chantilly, guipure, and mesh lace for wedding dresses. 
Bulk orders welcome, competitive prices.">

<!-- URL -->
/collections/lace-fabric-wholesale

<!-- H1 (页面只有一个) -->
<h1>Wholesale Lace Fabric for Fashion & Bridal</h1>

<!-- 正文首段 -->
<p>Looking for high-quality lace fabric wholesale? Fominte is a leading 
lace fabric supplier offering premium bridal and fashion lace in bulk...</p>

<!-- H2 小标题 -->
<h2>Our Lace Fabric Collections</h2>
<h2>Bulk Ordering & Minimum Quantities</h2>
<h2>Why Choose Fominte as Your Lace Supplier</h2>

<!-- 图片 Alt -->
<img src="..." alt="White chantilly lace fabric wholesale">
```

## Title 标签公式

对于不同页面类型，推荐的 Title 格式：

| 页面类型 | Title 公式 | 示例 |
|---------|-----------|------|
| 首页 | 品牌 + 核心业务 | `Fominte - Embroidery Fabric & Clothing Wholesale` |
| 分类页 | 品类词 + Wholesale/Supplier + 品牌 | `Lace Fabric Wholesale | Bulk Order - Fominte` |
| 产品页 | 产品名 + 品类 + 品牌 | `FL-2156 Chantilly Lace Fabric | Fominte` |
| pSEO页 | 产品 + 场景 + Wholesale + 品牌 | `Mesh Embroidery for Wedding Dress Wholesale | Fominte` |
| Blog | 文章标题 + 品牌 | `How to Choose Lace Fabric for Bridal Wear | Fominte` |

## 常见错误

| 错误 | 说明 |
|-----|------|
| ❌ Title 不包含关键词 | 最重要的位置不能空 |
| ❌ 关键词堆砌 | 不自然的重复会被惩罚 |
| ❌ H1 缺失或多个 | 每个页面只能有一个 H1 |
| ❌ URL 用中文或参数 | 应该用英文、连字符分隔 |
| ❌ 图片无 Alt | 错失排名机会 |

## Shopline 中如何设置

在 Shopline 后台：
- **Title**: 产品/页面编辑 → SEO 设置 → 页面标题
- **Meta Description**: 产品/页面编辑 → SEO 设置 → 描述
- **URL**: 产品/页面编辑 → SEO 设置 → URL 句柄
- **H1**: 通常是产品名/页面标题（需检查主题模板）

---

## 检查清单

每个重要页面上线前，确认：

- [ ] Title 包含目标关键词
- [ ] Title 长度 50-60 字符
- [ ] Meta Description 包含关键词 + 吸引点击
- [ ] H1 与 Title 呼应
- [ ] URL 简短、包含关键词
- [ ] 首段自然提到关键词
- [ ] 图片有描述性 Alt
