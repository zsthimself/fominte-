# 产品体系

本目录包含 Fominte 所有产品品类的详细信息。

## 业务线概览

```
Fominte
├── 绣花面料批发
│   ├── 网布绣花面料 (Mesh Embroidery)
│   ├── 钩针绣花面料 (Crochet Lace)
│   └── 时尚绣花面料 (Fashion Embroidery)
│
├── 成衣制造批发
│   ├── 仿手钩绣花成衣 (Crochet Clothing)
│   ├── 仿皮草成衣 (Faux Fur Clothing)
│   ├── 摇粒绒成衣 (Fleece Clothing)
│   └── 时尚定制礼服成衣 (Custom Formal Wear)
│
└── OEM/ODM 定制服务
    └── 可定制各类成衣（除工作制服外）
```
